If Windows warns you

Windows SmartScreen flags all new unsigned software downloaded from the internet. ST-SoftwareTool is completely safe. To install:

  1. Click More info on the blue SmartScreen screen
  2. Click Run anyway

Alternatively: right-click the downloaded file → Properties → tick Unblock → OK, then run it.

Requirements
  • Windows 10 / 11 (64-bit)
  • Administrator rights for full feature access
  • NVIDIA GPU recommended for GPU monitor
Why does Windows warn me?

Microsoft SmartScreen checks every downloaded .exe against a reputation database. New software with few downloads is flagged automatically regardless of whether it is safe.

ST-SoftwareTool does not connect to any external server, collect any data, or modify your system without your permission. All operations are local and visible.
